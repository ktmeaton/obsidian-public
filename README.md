My public Obsidian vaults.

## Vaults

- [Academic](academic/)

## Install

1. Clone repository.

	```bash
	git clone https://github.com/ktmeaton/obsidian-public.git
	```

1. Open folder as vault in Obsidian.
1. Enable all `Core Plugins` except:

  - Zettelkasten
  - Random Note
	- Publish
	- Sync

1. Turn OFF safe mode to enable community plugins.
